# Перечень артефактов проекта Lab-01-Stancenko

## Исходный код
- Program.cs — основной файл вызова модулей. Этот файл служит для связывания модулей между собой в центре узловой архитектуры разрабатываемого приложения.
- Modules/InputModule.cs — модуль ввода данных. Этот файл служит для реализации ввода корректных данных, для их будущей обработки.
- Modules/ValidationModule.cs — модуль проверки данных. Этот модуль организовывает определенную проверку данных, чтобы исклить какие либо ошибки и недопонимания.
- Modules/ProcessingModule.cs — модуль обработки и вывода данных. Этот модуль служит напрямую для таких целей, как обработка корректных данных, а также их корректный вывод.

## Вспомогательные файлы
- README.md — описание проекта
- .gitignore — исключения Git
